# comicd
comicd是一个漫画下载工具，指定漫画主页的url或者某个章节的url，就可以把对应的资源下载到本地。  
代码只在Python 3.6版本上测试过。


## 支持
* [腾讯动漫](http://ac.qq.com/)
* [网易漫画](https://manhua.163.com/)

## 安装
``` shell
$ git clone https://github.com/fancxxy/comicd.git ~/Works/comicd

$ cd ~/Works/comicd

$ python setup.py install
```
